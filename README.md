*소개

저에 첫번째로 배포한 웹사이트입니다. 이름에 약자 처럼 "모든 사용자들이 자신에 아이디어를 쉽고 빠르게 공유 할수있는 공간이란 의미"입니다. 즉, 웹 접근성(시각 청각 등 신체적으로 불편한 사람들, 영유아 및 노약자에 해당하는 사람들 등에 최적화한 사용자 인터페이스 및 사용자 경험을 고려하는것)을 최대한 웹앱에 적용하며 또한 사용하기도 보다 심플하고 편리하고 빠른 UI(사용자 인터페이스)
디자인을 위해 항상 고민하고 사용자들과의 대화에 얻은 의견들을 반영하여 종은 UI 디자인을 제공할수있도록 하죠 그리고 웹앱 안에서 자료를 쉽고 빠르게 공유 할수있도록 [인링크] 기능을 구성하였습니다.
